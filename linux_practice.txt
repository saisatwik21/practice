1. Initialize Git (if not already done)

   
   git init


2.Add remote repository URL
   (Replace `<your-username>` and `<repo-name>` with your GitHub details)

   
   git remote add origin https://github.com/<your-username>/<repo-name>.git


   > To check remote added:

   
   git remote -v
   

3. Stage your files
   Add all local files to staging:

   
   git add .


4. Commit your changes

  
   git commit -m "Initial commit"
   

5. Push code to GitHub
   For the first push, use:

   
   git branch -M main     # rename current branch to main (GitHub default)
   git push -u origin main
   

   After this, for future changes, you can simply run:

   
   git push
